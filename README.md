## 管理员登录系统

使用vite+pinia+tailwindcss+element-plus+mock实现管理员登录系统

![node](https://img.shields.io/badge/node-%3E=18.0.0-brightgreen)

登录界面如下
![登录界面](./src/assets/images/Snipaste01.png)

## 安装

pnpm
```shell
pnpm install
```

yarn 
```shell
yarn install
```

npm 
```shell
npm install
```

## 运行
``` shell
npm run dev
```

## 打包
```shell
npm run build
```
